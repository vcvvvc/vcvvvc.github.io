Dell 灵越 7590 install macos Ventura

Jan 17, 2021
16 mins read
Dell Mac Ventura MacOS

2026-03-02更新

现在OC已经配置好了kext，可以直接用。

蓝牙/wifi/hdmi可用，麦克风不可能，此问题无解。

镜像下载

搜素：黑果小兵

Start

//OC
https://github.com/vcvvvc/Dell-Inspiron-7590-Ventura-OpenCore.git

//Diskgenius
官网自行下载最新版

//balennaEther
官网自行下载最新版

//OpenCore Configurator
OpenCore Configurator v26.69.0.1 支持OC v0.9.1

1. 打开OC文件夹 对应的1080p/4k文件需要改名为config.plist

2. balennaEther烧录dmg镜像文件到U盘

3. 打开Diskgenius 进入U盘的ESP分区文件夹 删除Clover文件夹 然后将OC文件夹拖入

4. 打开U盘ESP分区中的EFI 将Diskgenius程序文件夹拖进去 备用 以免出问题

提前准备好安装空间建议300g 至少200g

5. 用Diskgenius分ESP区 建议400m

6. 双系统 先安装好windows， 然后保存好windows efi

安装

1. 重启F2 关闭Secure Boot安全启动 检查磁盘模式为ACHI 教程网络搜

2. 进入Boot Sequence ‘add option’ 添加一个引导 选择启动文件为自己的U盘EFI(找到EFI/OC/OpenCore.efi这个文件)

3. 重启F12 选择刚才新建的启动项 选择Install mac Ventura

4. 选择磁盘工具先抹盘 选择你提前分好的空间 选择格式为APFS 抹不掉就先做成exft 再抹一次

5. 左上角退出磁盘工具继续安装 然后自动重启 再次选mac启动项

6. 重启f12 选择install mac 本步骤需要重复几次 直到进入系统设置 创建用户的步骤才算完成

  • Siri不启用 iCloud先不要登陆 先创建好一个本地账户

7.1 重启f12 选择winPE, 在我的电脑 EFI中找到Diskgenius 打开程序将U盘的EFI文件复制到桌面, 此时可以把win10的efi加回去

7.2 选择磁盘的ESP-EFI将OC拖入 / 或者后面挂载后拖入

8.1 Disk这个软件可以设置启动项,或者重启F2, Boot Sequence add option 这里我命名为macos 选择磁盘中EFI\OC\OpenCore.efi

8.2 把选项置顶。开机自动进入 选择macos

8.3 文章最后网友录制的B站教程

三码洗白

电脑提前退出账号

删除登陆的电脑设备 手机-设置-最上方进去-找到型号删除掉

一定要找对自己的型号 dell7590选择MacPro 16.4, 然后检查序列号，确保查询不到，如果查询到设备就重新生成一个。

重启登陆 正常使用功能

三码洗白后

1、 登陆Apple id，关闭同步 2、关闭siri的聚焦中部分用不到的文件，不然同步过多风扇起飞

关闭SIP(可选)

增加一个csr-active-config填写值FF0F0000
