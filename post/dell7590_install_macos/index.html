Dell7590 install macos

Jan 17, 2021
Take 1 min read
DellMacos

First

接近完美 可以用intel网卡连接wifi 速度还不错

装好之后win10引导没了 反正我也不用 其他问题只发现内置麦克风无法使用 Siri不能用 蓝牙未启动 也就没测试

U盘16g 再来个小内存U盘或者有网线联网
分出一块磁盘空间至少100G

Start

//OC
git clone https://github.com/Pinming/Dell-Inspiron-7590-Hackintosh-Opencore

//WIFI 这四个可以提前下好存另一个U盘或者网线下
https://github.com/OpenIntelWireless/HeliPort/releases
https://github.com/OpenIntelWireless/itlwm/releases/
https://mackie100projects.altervista.org/download-clover-configurator/
https://mackie100projects.altervista.org/download-opencore-configurator/

//Diskgenius
官网自行下载最新版

//balennaEther
官网自行下载最新版

先把Dell-Inspiron-7590-Hackintosh-Opencore重命名为OC

打开OC文件夹 根据屏幕改名1080P的屏幕使用config-1080p.plist需改名成config.plist 如果你是4K屏 不需要修改

balennaEther烧录dmg镜像文件到U盘

打开Diskgenius 进入U盘的ESP分区文件夹 删除Clover文件夹 然后将OC文件夹拖入

打开U盘ESP的EFI 将Diskgenius程序文件夹拖进去 备用 以免出问题

提前准备好安装空间100g

再创建出一个300MB的空间 用Diskgenius做成ESP

继续

重启F2 关闭Secure Boot安全启动 检查磁盘模式为ACHI 百度搜

之后进入Boot Sequence ‘add option.?'(忘了名字了add**) 添加一个 选择启动文件为自己的U盘EFI(找到EFI/OC/OpenCore.efi这个文件)

重启F12 选择刚才新建的启动项 选择Install mac

进去之后选择磁盘工具先抹盘 选择你提前分好的空间 命名个macos就行 选择格式为APFS 抹不掉就先做成exft 再抹一次

退出磁盘工具继续安装 然后自动重启 再次选mac启动项

Siri不启用 iCloud先不要登陆 先创建好一个本地账户

这是网上找的

重启电脑，进入BIOS，把刚刚改过的设置全部改回来（启动顺序，磁盘模式，安全启动）

这三件套一定要改回来，不然回不到原来的Windows系统！

这时候我这边win10进不去了 直接开机选择PE 在磁盘找到Diskgenius 打开程序 将U盘的OC复制到桌面 之后 打开磁盘的ESP 将OC拖到EFI文件之下

重启F2 Boot Sequence add option 选择磁盘ESP 大概路径是这个EFI\OC\OpenCore.efi

重启F12 进入这个启动项 进来了就OK了

Wifi设置

需要提前下或者网线下载的 两个软件先装好

安装HeliPort 然后设置-用户与群组-登录项 添加HeliPort

打开Clover Configurator 选择左侧栏目的挂载分区 然后右边EFI分区 挂载分区

打开分区 进去OC文件夹 config.plist右键选择Clover Configurator打开

左侧机型设置 右侧内存信息网上一点点 然后最右侧 选择macpro15,3 生成序列号 跟Smuuid 多点几次生成 关闭点保存

config.plist右键再用Opencore打开 左侧内核设置把itlwm.kext拖进右边 然后启用点上 重启即可 开机就有wifi了

安装成功

三码洗白

电脑提前退出账号

删除登陆设备 手机-设置-最上方进去-找到型号删除掉

1. 安装Clover Configurator

#一定要找对自己的型号 dell7590选择``MacPro 15.3``
2. 选择合适的SMBIOS ID

1. 生成UUID 复制到图中SmUUID中

2. 点击生成新的序号

复制序列号到网址检查 查询不到就是可以用

https://checkcoverage.apple.com/cn/zh

重启登陆 正常使用功能
