Dell7590 install macos

Jan 17, 2021
22 mins read
Dell Mac

2021-12-13更新

系统为big sur 11.4

接近完美 可以用intel网卡连接wifi 速度还不错

已知麦克风、隔空投送不能用

镜像下载

搜素：黑果小兵

big sur用11.4

重要前提

如果你没有网线，并且没有其他PC设备联网 你需要提前下载这些

https://github.com/OpenIntelWireless/itlwm/releases/ #AirportItlwm
https://5435.lanzouo.com/isRfBxjxq4f #四叶草+occ

Start

//OC
git clone https://github.com/Pinming/Dell-Inspiron-7590-Hackintosh-Opencore

//Diskgenius
官网自行下载最新版

//balennaEther
官网自行下载最新版

1. 先把Dell-Inspiron-7590-Hackintosh-Opencore重命名为OC

2. 打开OC文件夹 根据屏幕改名1080P的屏幕使用config-1080p.plist需改名成config.plist 如果你是4K屏 不需要修改

3. balennaEther烧录dmg镜像文件到U盘

4. 打开Diskgenius 进入U盘的ESP分区文件夹 删除Clover文件夹 然后将OC文件夹拖入

5. 打开U盘ESP分区中的EFI 将Diskgenius程序文件夹拖进去 备用 以免出问题

提前准备好安装空间建议300g 至少200g

6. 用Diskgenius分ESP区 建议400m

7. 双系统需要保存好windows efi

安装

1. 重启F2 关闭Secure Boot安全启动 检查磁盘模式为ACHI 百度搜

2. 进入Boot Sequence ‘add option’ 添加一个引导 选择启动文件为自己的U盘EFI(找到EFI/OC/OpenCore.efi这个文件)

3. 重启F12 选择刚才新建的启动项 选择Install mac big sur

4. 选择磁盘工具先抹盘 选择你提前分好的空间 命名个macos就行 选择格式为APFS 抹不掉就先做成exft 再抹一次

5. 左上角退出磁盘工具继续安装 然后自动重启 再次选mac启动项 (此时我读条完毕 一直在转圈 等了5分钟 我手动强制关机)

6. 重启f12 选择install mac 本步骤需要重复几次 直到进入系统设置 创建用户的步骤才算完成

  • Siri不启用 iCloud先不要登陆 先创建好一个本地账户

7.1 重启f12 选择winPE, 在我的电脑 EFI中找到Diskgenius 打开程序将U盘的EFI文件复制到桌面, 此时可以把win10的efi加回去

7.2 选择磁盘的ESP-EFI将OC拖入

8.1 Disk这个软件可以设置启动项,或者重启F2, Boot Sequence add option 这里我命名为macos 选择磁盘中EFI\OC\OpenCore.efi

8.2 把选项置顶。开机自动进入 选择macos

Wifi设置

下载–点击: 前提文件

挂载EFI

将AirportItlwm_xx.kext加入efi-oc-kexts，再拖入occ内核设置

安装成功

三码洗白

电脑提前退出账号

删除登陆的电脑设备 手机-设置-最上方进去-找到型号删除掉

一定要找对自己的型号 dell7590选择MacPro 16.4

重启登陆 正常使用功能

关闭SIP(可选)

增加一个csr-active-config填写值FF0F0000
