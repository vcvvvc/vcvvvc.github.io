KDJ

KDJ图形是一种用于分析股票或期货市场的技术指标，它由三条曲线组成：K线、D线和J线。KDJ图形可以反映市场的趋势、强弱、超买超卖和拐点等信息，从而帮助投资者判断买卖时机。下面是一些关于KDJ图形的基本知识：

  • KDJ图形的计算方法是利用最高价、最低价和收盘价来求出RSV值，然后用平滑移动平均法来计算K值、D值和J值。具体的公式如下¹：
 RSV = (收盘价 - 最近N日最低价) / (最近N日最高价 - 最近N日最低价) * 100%
 K = 2/3 * 前一日K + 1/3 * 当日RSV
 D = 2/3 * 前一日D + 1/3 * 当日K
 J = 3 * D - 2 * K

其中，N是一个参数，一般取9或14，表示计算RSV值时的周期长度。如果没有前一日的K值和D值，可以用50代替。

  • KDJ图形的取值范围是0到100，其中50是多空均衡线，80以上是超买区，20以下是超卖区。K线是快速确认线，D线是慢速主干线，J线是方向敏感线。一般来说，K线和D线的交叉是KDJ图形的主要信号，J线的变化是KDJ图形的辅助信号²。

    • 当K线从下向上穿越D线，形成金叉，表示市场由弱转强，是买入的信号。金叉的位置越低越好，尤其是在超卖区形成的金叉，更有利于抓住市场的反弹机会³。
    • 当K线从上向下穿越D线，形成死叉，表示市场由强转弱，是卖出的信号。死叉的位置越高越好，尤其是在超买区形成的死叉，更有利于抓住市场的回调机会³。
    • 当J线从零轴以下突破K线和D线，表示市场的弱势逐步减弱，是买入的信号。当J线从零轴以上向下掉头，表示市场的强势逐步减弱，是卖出的信号⁴。
    • 当J线大于100或小于0，表示市场处于极端的超买或超卖状态，可能会出现反转的机会。但是，反转的方式有两种：一种是实质性的反转，即价格发生明显的上涨或下跌；另一种是修复性的反转，即价格在高位或低位平移，使J线缓慢回归正常范围⁵。因此，不能单纯地根据J线的超买超卖来判断市场的走势，还要结合其他指标和形态来综合分析。

  • KDJ图形的形态运行方式有很多，其中最重要的有三种：交叉、背离和钝化。

    • 交叉是指K线和D线的相交，可以分为单次交叉和多次交叉。单次交叉就是前面提到的金叉和死叉，是KDJ图形的基本信号。多次交叉是指K线和D线在一定的区间内反复交叉，形成一些特殊的形态，如W底、M头、头肩顶、头肩底、双顶、双底等。这些形态可以反映市场的转折点和趋势的延续，是KDJ图形的高级信号。
    • 背离是指KDJ图形的走势方向与价格的走势方向不一致，可以分为顶背离和底背离。顶背离是指价格创出新高，而KDJ图形没有创出新高，表示上涨的动能不足，多头力量衰竭，有下跌的趋势。底背离是指价格创出新低，而KDJ图形没有创出新低，表示下跌的动能不足，空头力量衰竭，有上涨的趋势。背离是KDJ图形的重要信号，可以帮助投资者发现市场的顶部和底部。
    • 钝化是指KDJ图形在高位或低位的变化幅度很小，而价格却在继续上涨或下跌，可以分为高位钝化和低位钝化。高位钝化是指KDJ图形在超买区域，价格继续上涨，而KDJ图形变化不明显，甚至向下。低位钝化是指KDJ图形在超卖区域，价格继续下跌，而KDJ图形变化不明显，甚至向上。钝化是KDJ图形的特殊信号，表示市场的强势或弱势难以改变，但也可能是市场即将反转的前兆。

买入信号

  • KDJ指标的买入信号主要有以下几种：
    • K线和D线在低位出现金叉，即K线从下向上穿越D线，表示市场由弱转强，是买入的信号。如果金叉出现在50以下的低位，或者在超卖区间内，信号更强烈。
    • D线和J线进入超卖区间，即20以下，表示市场已经过度卖空，有反弹的可能，是买入的信号。如果D线从超卖区间向上突破20，或者J线从零轴以下向上突破K线和D线，信号更强烈。
    • K线和股价出现底背离，即股价创出新低，而K线没有创出新低，反而出现一底比一底高的上涨走势，表示市场的下跌动能减弱，有反转的可能，是买入的信号。
  • KDJ指标的买入时机可以根据不同的目的和风险偏好来选择，一般来说，可以分为以下几种情况：
    • 如果想尝试买入，可以在K线和D线在低位出现金叉时，先少量建仓，或者在D线和J线进入超卖区间时，先少量建仓。这样的买入信号相对较为准确，但也有可能是假信号，所以要控制好仓位，观察后续的走势。
    • 如果想大量买入，可以在K线和D线在低位出现金叉后，持续向上移动，突破50的多空均衡线时，积极加仓，或者在D线和J线从超卖区间向上突破20时，积极加仓。这样的买入信号相对较为强烈，说明市场的上涨动能增强，股价有持续上涨的趋势。
    • 如果想抓住反转的机会，可以在K线和股价出现底背离时，积极买入。这样的买入信号相对较为稀有，但也很有效，说明市场的下跌力量衰竭，有上涨的趋势。

卖出信号

  • KDJ指标的卖出信号主要有以下几种：
    • K线和D线在高位出现死叉，即K线从上向下穿越D线，表示市场由强转弱，是卖出的信号。如果死叉出现在50以上的高位，或者在超买区间内，信号更强烈。
    • D线和J线进入超买区间，即80以上，表示市场已经过度买入，有回调的可能，是卖出的信号。如果D线从超买区间向下突破80，或者J线从零轴以上向下突破K线和D线，信号更强烈。
    • K线和股价出现顶背离，即股价创出新高，而K线没有创出新高，反而出现一峰比一峰低的下跌走势，表示市场的上涨动能减弱，有反转的可能，是卖出的信号。
  • KDJ指标的卖出时机可以根据不同的目的和风险偏好来选择，一般来说，可以分为以下几种情况：
    • 如果想尝试卖出，可以在K线和D线在高位出现死叉时，先少量减仓，或者在D线和J线进入超买区间时，先少量减仓。这样的卖出信号相对较为准确，但也有可能是假信号，所以要控制好仓位，观察后续的走势。
    • 如果想大量卖出，可以在K线和D线在高位出现死叉后，持续向下移动，跌破50的多空均衡线时，积极减仓，或者在D线和J线从超买区间向下突破80时，积极减仓。这样的卖出信号相对较为强烈，说明市场的下跌动能增强，股价有持续下跌的趋势。
    • 如果想抓住反转的机会，可以在K线和股价出现顶背离时，积极卖出股票。这样的卖出信号相对较为稀有，但也很有效，说明市场的上涨力量衰竭，有下跌的趋势。

MACD

MACD图形是一种用于分析股票或期货市场的技术指标，它由三个部分组成：DIF线、DEA线和MACD柱。DIF线是快速移动平均线和慢速移动平均线的差值，DEA线是DIF线的平滑移动平均线，MACD柱是DIF线和DEA线的差值的柱状图。MACD图形可以反映市场的趋势、强弱、转折和背离等信息，从而帮助投资者判断买卖时机。下面是一些关于MACD图形的基本知识：

  • MACD图形的计算方法是利用收盘价的指数移动平均线来求出DIF值和DEA值，然后用柱状图来表示它们的差值。具体的公式如下¹：
 DIF = EMA(close,12) - EMA(close,26)
 DEA = EMA(DIF,9)
 MACD = DIF - DEA

其中，EMA是指数移动平均线，close是收盘价，12和26是常用的参数，表示计算DIF值时的周期长度，9是常用的参数，表示计算DEA值时的周期长度。如果没有前一日的DIF值和DEA值，可以用0代替。

  • MACD图形的取值范围是无限的，但一般在-10到10之间。零轴是多空分界线，当DIF线和DEA线在零轴以上时，表示多头市场，当DIF线和DEA线在零轴以下时，表示空头市场。MACD柱的颜色是根据DIF线和DEA线的差值的正负来确定的，当差值为正时，柱状图为红色，表示上涨力量，当差值为负时，柱状图为绿色，表示下跌力量。

  • MACD图形的主要信号是DIF线和DEA线的交叉，可以分为金叉和死叉。金叉是指DIF线从下向上穿越DEA线，表示市场由弱转强，是买入的信号。死叉是指DIF线从上向下穿越DEA线，表示市场由强转弱，是卖出的信号。一般来说，金叉和死叉的位置越接近零轴，信号越强，位置越远离零轴，信号越弱。²

  • MACD图形的辅助信号是MACD柱的变化，可以分为放大和收缩。放大是指MACD柱的长度逐渐增加，表示市场的动能增强，趋势延续的可能性大。收缩是指MACD柱的长度逐渐减少，表示市场的动能减弱，趋势转变的可能性大。一般来说，当MACD柱由红变绿或由绿变红，表示市场的转折信号。³

  • MACD图形的特殊信号是背离，可以分为顶背离和底背离。背离是指MACD图形的走势方向与价格的走势方向不一致，反映了市场的隐藏力量。顶背离是指价格创出新高，而MACD图形没有创出新高，表示上涨的动能不足，多头力量衰竭，有下跌的趋势。底背离是指价格创出新低，而MACD图形没有创出新低，表示下跌的动能不足，空头力量衰竭，有上涨的趋势。背离是MACD图形的重要信号，可以帮助投资者发现市场的顶部和底部。⁴

  • MACD图形的形态运行方式有很多，其中最重要的有四种：佛手向上、小鸭出水、漫步青云和天鹅展翅。⁵

    • 佛手向上是指在零轴上，DIF线和DEA线出现金叉后，又出现死叉，但不下穿零轴，几日后在零轴上再次出现金叉，形成二次金叉。这种形态是主力洗盘所为，短暂盘整后，股价将上扬，是积极买入的信号。
    • 小鸭出水是指在零轴下，DIF线处于较低位置且不再下行，开始呈横向水平移动，而此时股价继续处于下跌状态，这已开始预示股价见底，一旦DIF线平移结束，开始向上抬头形成金叉，是买入股票的最佳时机。
    • 漫步青云是指在零轴上，DIF线和DEA线出现金叉后，持续向上移动，且MACD柱不断放大，形成一片红海，表示股市处于强势之中，股价将继续上涨，是持股待涨或加仓买入的信号。
    • 天鹅展翅是指在零轴下，DIF线和DEA线出现金叉后，持续向下移动，且MACD柱不断放大，形成一片绿海，表示股市处于弱势之中，股价将继续下跌，是持币观望或卖出股票的信号。

买入信号

  • MACD是一种常用的技术分析指标，它是由快速移动平均线（DIF）和慢速移动平均线（DEA）以及它们之间的差值（MACD柱线）组成，可以反映市场的趋势、强弱、转折等信息，从而提供买卖的参考。
  • MACD的买入信号主要有以下几种：
    • DIF线和DEA线在零轴下方出现金叉，即DIF线从下向上穿越DEA线，表示市场由弱转强，是买入的信号。如果金叉出现在零轴附近，或者在超卖区间内，信号更强烈。
    • DIF线和DEA线在零轴上方出现金叉，即DIF线从下向上穿越DEA线，表示市场由强转更强，是买入的信号。如果金叉出现在50以上的高位，或者在超买区间内，信号更强烈。
    • DIF线和DEA线在零轴上方运行一段时间后，DIF线回调到DEA线附近，然后再次向上反转，形成空中缆绳形态，表示市场经过短暂的调整后，继续上涨，是买入的信号。
    • DIF线和DEA线在零轴下方运行一段时间后，DIF线回调到DEA线附近，然后再次向上反转，形成海底电缆形态，表示市场经过长期的探底后，开始反弹，是买入的信号。
    • DIF线和股价出现底背离，即股价创出新低，而DIF线没有创出新低，反而出现一底比一底高的上升走势，表示市场的下跌动能减弱，有反转的可能，是买入的信号。
  • MACD的买入时机可以根据不同的目的和风险偏好来选择，一般来说，可以分为以下几种情况：
    • 如果想尝试买入，可以在DIF线和DEA线在零轴下方出现金叉时，先少量建仓，或者在DIF线和DEA线在零轴上方出现金叉时，先少量建仓。这样的买入信号相对较为准确，但也有可能是假信号，所以要控制好仓位，观察后续的走势。
    • 如果想大量买入，可以在DIF线和DEA线在零轴下方出现金叉后，持续向上移动，突破零轴时，积极加仓，或者在DIF线和DEA线在零轴上方出现金叉后，持续向上移动，突破50的多空均衡线时，积极加仓。这样的买入信号相对较为强烈，说明市场的上涨动能增强，股价有持续上涨的趋势。
    • 如果想抓住反转的机会，可以在DIF线和股价出现底背离时，积极买入股票。这样的买入信号相对较为稀有，但也很有效，说明市场的下跌力量衰竭，有上涨的趋势。
