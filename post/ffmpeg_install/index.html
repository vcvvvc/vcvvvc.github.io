Ubuntu20 install FFmpeg

Jul 27, 2020
1 min read
LinuxFFmpegUbuntu

Start Download FFmpeg

Install tool

1. install yasm

sudo apt-get install yasm

if not installed yasm, os will output “nasm/yasm not found or too old. Use –disable-x86asm for a crippled build.”

2. install SDL2

SDL2-2.0.12(stable)

tar -xvf SDL2-2.0.12.tar.gz
cd SDL2-2.0.12
./configure
sudo make -j4
sudo make install

Install FFmpeg

1. git clone https://git.ffmpeg.org/ffmpeg.git ffmpeg

cd ffmpeg/
./configure
sudo make -j4
sudo make install

3. Check

ffmpeg -version
ffplay -version
