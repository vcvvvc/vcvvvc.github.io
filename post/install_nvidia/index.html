Ubuntu20 Install Nvidia drivers

Jul 29, 2020
1 min read
Linux drivers Ubuntu Nvidia

0.Ready work

Disable BIOS secure boot，choose disable .

1. Uninstall old Nvidia drivers

sudo apt-get remove --purge nvidia*

2. Add rules

vim /etc/modprobe.d/blacklist.conf

1. blacklist nouveau 
2. options nouveau modeset=0

3. Verification

After reboot, use the following command no return is successful

sudo update-initramfs -u
lsmod | grep nouveau

4. Choose Navidia drivers version

  • 1.Use command: ubuntu-drivers devices
  • 2.Nvidia Official website

Choose stable version and install

sudo add-apt-repository ppa:graphics-drivers/ppa
sudo apt-get update
sudo apt-get install nvidia-driver-***

5. End

reboot
nvidia-smi

Open warehouse => additional drivers => choose you drivers .
